# Scripts Module

This directory contains runnable scripts for CEREBRUM testing and examples.

## Contents

- **`run_all_examples.py`**: Execute all example scripts
- **`run_all_tests.py`**: Run the complete test suite
- **`run_tests.py`**: Configurable test runner
- **`test_all_cases.py`**: Test all case transformations

## Usage

```bash
# Run all tests
python -m src.scripts.run_all_tests

# Run all examples
python -m src.scripts.run_all_examples
```

# Examples

This directory contains standalone example scripts demonstrating CEREBRUM's capabilities.

## Contents

| File | Purpose |
| ---- | ------- |
| `__init__.py` | Module exports |
| `animal_agent.py` | Animal agent with case transformations |
| `streamlined_animal_agent.py` | Simplified animal agent |
| `environment.py` | Environment for agent simulations |
| `insect_simulation_example.py` | Basic insect model with Active Inference |
| `comprehensive_insect_simulation.py` | Advanced multi-factor insect simulation |
| `simplified_insect_simulation.py` | Lightweight insect simulation |
| `insect_visualization_example.py` | Insect-specific visualizations |
| `thermostat.py` | Thermostat Active Inference example |
| `hvac_system.py` | HVAC system Active Inference example |

## Running

```bash
# From repository root
python -m src.examples.insect_simulation_example
python -m src.examples.animal_agent
python -m src.examples.thermostat
```

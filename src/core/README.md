# Core Components

This directory contains the foundational classes and engines for CEREBRUM.

## Key Files

- **`model.py`**: Defines the base `Model` class that all cognitive models must inherit from.
- **`active_inference.py`**: The Active Inference engine implementing free energy minimization.
- **`neural_network.py`**: Neural network components integrated with the linguistic case system.
- **`model_registry.py`**: System for registering and retrieving model types.
